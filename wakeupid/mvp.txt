# ⏰ MemeAlarm (MVP + Meme UI Lite)

## 📌 Overview

MemeAlarm ek fun alarm app hai jo user ko uthne ke liye force karta hai using **math challenge + meme content**.

Is MVP version me hum basic functionality ke saath thoda **fun + personality** add karenge.

---

## 🚀 MVP Goal

* Alarm system working ho
* Math challenge se alarm band ho
* Basic meme messages + UI ho
* Simple but fun experience mile

---

## 📱 Features (MVP Lite)

### ✅ Alarm System

* Set alarm
* Alarm ring screen

### ✅ Math Challenge

* Random question
* Correct answer → alarm off

### 😂 Meme Mode (Lite)

Alarm screen pe random funny text show hoga:

Examples:

* "Uth ja bhai, duniya jeetni hai 😤"
* "Snooze mat kar… tu Elon Musk nahi hai"
* "Sleep is temporary, regret is permanent 💀"
* "Aaj bhi nahi utha to kuch nahi badlega"

---

## 🎨 UI Design (Simple + Meme)

### Theme

* Dark background
* Neon accent (green / purple)
* Big bold text

### Screens

1. **Home Screen**

   * Alarm list
   * Add alarm button

2. **Alarm Ring Screen**

   * Big time display
   * Meme text
   * Vibrating UI (animation later)

3. **Challenge Screen**

   * Math question
   * Input box

## ⚙️ Setup

```bash id="a1b2c3"
npx create-expo-app MemeAlarm
cd MemeAlarm
npm install @react-navigation/native
npm install @react-navigation/stack
npm install zustand
npm install @react-native-async-storage/async-storage
npm install expo-notifications
```

Run:

```bash id="d4e5f6"
npx expo start
```

## 🧠 Core Logic

### Alarm Flow

1. User alarm set kare
2. Alarm trigger ho
3. Alarm Ring Screen open ho
4. Random meme text show ho
5. User → "Stop Alarm" click kare
6. Math challenge open ho
7. Correct answer → alarm band
